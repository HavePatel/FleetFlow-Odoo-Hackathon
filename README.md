# 🚚 FleetFlow – Fleet Management System

FleetFlow is a full-stack fleet management system built during the Odoo Hackathon.

It allows fleet managers to manage:
- Vehicles
- Drivers
- Trips (Dispatch & Complete)
- Maintenance records
- Live dashboard statistics

---

## 🏗 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios

### Backend
- FastAPI
- SQLAlchemy
- Pydantic
- Uvicorn

### Deployment
- Backend deployed on Render
- Frontend running locally (or can be deployed to Vercel/Netlify)
